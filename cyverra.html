Enter your Gmail details so every contact form submission is emailed to you instantly. Use a Gmail App Password (not your normal password).

How to get a Gmail App Password:
1. Go to myaccount.google.com/security
2. Enable 2-Step Verification if not already on
3. Search for "App Passwords" → create one → select Mail
4. Copy the 16-character password it generates
✓ Email settings saved! You will now receive notifications for every message.
